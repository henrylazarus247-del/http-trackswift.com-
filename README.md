# TrackSwift - Package Tracking Website

A modern package tracking website to monitor shipment status in real-time.

## Features
- Real-time package status tracking
- Interactive tracking timeline
- Responsive design
- Support for multiple tracking statuses

## Project Structure
```
.
├── index.html          # Main page
├── css/
│   └── styles.css      # Styling
├── js/
│   ├── app.js          # Main application logic
│   └── tracking.js     # Tracking status handler
├── data/
│   └── tracking-data.json  # Sample tracking data
└── README.md           # This file
```

## Getting Started
1. Clone the repository
2. Open `index.html` in your browser
3. Enter a tracking number to view package status

## Status Types
- ✓ Package Received
- ✓ Processing
- ✓ In Transit
- ○ Out for Delivery
- ○ Delivered
